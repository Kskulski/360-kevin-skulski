Final Project Proposal

Reflection Recorder

Created By Kevin Skulski

Last updated: 04/27/2021

Purpose:

The goal of this application is to allow for users to enter a reflection which is stored in a database and can later be retrieved for review.

Prioritized Features:

  1. Provide user interface for logging in and submitting a reflection.
  2. Successfully store the reflection in the database.
  3. Allow for the user to review the reflection.

Non-Prioritized Features:

  1. Allow the class professor to review all reflections.
  2. Include the ability for the professor to comment on the reflections.

Languages and Platforms:

  • HTML/CSS/JavaScript
  • MongoDB
  • Azure
  • Azure Functions

Implementation of Class Learning:

  • Front end web development
  • React
  • Hosting using Azure

Implementation of Material Learned Beyond Class:
